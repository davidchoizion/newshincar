
# Lotto Number Generator

## Overview

This project is a simple web application that generates random lottery numbers. The user can click a button to get a new set of 6 unique numbers between 1 and 45.

## Implemented Features

*   **Lottery Number Generation:** Generates 6 unique random numbers.
*   **User Interface:** A clean and simple interface to display the numbers and a button to generate them.
